# run_data

Programming assignment for Coursera "Getting and Cleaning Data" course.

## Instructions

The file run_analysis.R should be in the same directory as the following files:

* ./activity_labels.txt
* ./features.txt

And in the parent directory of /test and /train:

* ./test/X_test.txt
* ./test/y_test.txt
* ./test/subject_test.txt
* ./train/X_train.txt
* ./train/y_train.txt
* ./train/subject_train.txt

The source data files are downloaded from the following location:

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

In order to process the data the following commands should be run in R or RStudio:

```R
> source("run_analysis.R")
> init()
```

run_tidy.txt will be saved in the same directory as run_analysis
